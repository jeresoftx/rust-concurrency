# Soluciones

Aqui viven las soluciones ejecutables de ejercicios de niveles 1 a 3. Cada
solucion debe tener la misma calidad que el codigo principal: clara, probada y
documentada cuando la decision no sea obvia.

Los ejercicios de nivel 4 no tienen una unica solucion canonica; se documentan
como discusion de enfoques y tradeoffs dentro del capitulo.
