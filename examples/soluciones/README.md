# Soluciones

Aquí viven las soluciones ejecutables de ejercicios de niveles 1 a 3. Cada
solución debe tener la misma calidad que el código principal: clara, probada y
documentada cuando la decisión no sea obvia.

Los ejercicios de nivel 4 no tienen una única solución canónica; se documentan
como discusión de enfoques y tradeoffs dentro del capítulo.
